# Optimization Plan – Refactor Scripts & Shared Utilities

**Goal**: Consolidate duplicated logic across the `scripts/` directory, improve maintainability, readability, and idempotency while preserving existing behavior.

## 1. Inventory & Analysis
- List all scripts in `scripts/`.
- Identify repeated blocks:
  - Root‑user check (`if [[ $EUID -ne 0 ]]; ...`).
  - Package detection & installation (`dpkg -s`, `apt install`).
  - VFIO module checks and `/etc/modules` updates.
  - GRUB manipulation (`/etc/default/grub` updates).
  - Logging / status messages.
  - `set -euo pipefail` pattern.
  - Environment‑variable handling for optional overrides (e.g., `FORCE_REBUILD`, `NETWORK_MODE`).

## 2. Create Shared Utility Library
Add `scripts/lib/utils.sh` containing reusable helpers:
```bash
#!/usr/bin/env bash
set -euo pipefail

# Logging helpers with timestamps
log() { echo "$(date '+%Y-%m-%d %H:%M:%S') $*"; }
log_info()  { log "INFO: $*"; }
log_warn()  { log "WARN: $*"; }
log_error() { log "ERROR: $*" >&2; }

# Ensure script runs as root
ensure_root() {
  if [[ $EUID -ne 0 ]]; then
    log_error "Run script as root"
    exit 1
  fi
}

# Install missing apt packages (idempotent)
install_missing_packages() {
  local packages=("$@")
  local missing=()
  for pkg in "${packages[@]}"; do
    dpkg -s "$pkg" >/dev/null 2>&1 || missing+=("$pkg")
  done
  if (( ${#missing[@]} )); then
    log_info "Installing missing packages: ${missing[*]}"
    apt-get update -y
    apt-get install -y "${missing[@]}"
  else
    log_info "All required packages are already installed"
  fi
}

# Ensure a line exists in a file (e.g., /etc/modules)
ensure_line_in_file() {
  local line="$1" file="$2"
  grep -qxF "$line" "$file" || echo "$line" >> "$file"
}

# Update GRUB cmdline idempotently
update_grub_cmdline() {
  local required_param="$1"
  local grub_file="/etc/default/grub"
  local current=$(grep -E '^GRUB_CMDLINE_LINUX_DEFAULT=' "$grub_file" || true)
  local args=""
  if [[ $current =~ ^GRUB_CMDLINE_LINUX_DEFAULT=\"(.*)\"$ ]]; then
    args="${BASH_REMATCH[1]}"
  fi
  # Remove existing iommu params
  args=$(echo "$args" | sed -E 's/(intel_iommu=on|amd_iommu=on)//g; s/(^| )iommu=pt( |$)/ /g' | xargs)
  # Add required params
  if [[ -n $args ]]; then
    args="$required_param $args iommu=pt"
  else
    args="$required_param iommu=pt"
  fi
  sed -i "s|^GRUB_CMDLINE_LINUX_DEFAULT=.*$|GRUB_CMDLINE_LINUX_DEFAULT=\"$args\"|" "$grub_file"
  log_info "Updated GRUB cmdline: $args"
  update-initramfs -u -k all
  update-grub
}
```
Make the file executable (`chmod +x`) and add a brief header comment.

## 3. Refactor Individual Scripts
For each script (`01-install-proxmox-tools.sh`, `02-enable-iommu.sh`, …):
1. Insert at the top:
```bash
#!/usr/bin/env bash
source "$(dirname "${BASH_SOURCE[0]}")/lib/utils.sh"
ensure_root
```
2. Replace duplicated sections with calls to the utility functions:
   - Replace package‑check loop with `install_missing_packages curl wget vim git htop jq unzip gnupg lsb-release qemu-guest-agent net-tools dnsutils pciutils usbutils zip tar`.
   - Replace module‑check loop with `ensure_line_in_file vfio /etc/modules` etc.
   - Replace GRUB handling with `update_grub_cmdline "$required_param"`.
   - Replace `echo` status messages with `log_info`, `log_warn`, `log_error`.
3. Remove now‑redundant `set -euo pipefail` lines (already in utils.sh).
4. Keep script‑specific variables (e.g., `FORCE_REBUILD`, `NETWORK_MODE`).
5. Replace final `echo "DONE"` with `log_info "DONE"`.

## 4. Verify Idempotency & Behavior
- Run each script on a fresh VM (or disposable container) to ensure it still performs its task.
- Run a script twice consecutively; confirm no duplicate entries in `/etc/modules` or GRUB.
- Check that logs are timestamped and clearer.

## 5. Documentation Update
- Add a `README.md` section in `scripts/` describing the new `utils.sh` library, its exported functions, and how to source it.
- Add inline comments where complex logic was abstracted.
